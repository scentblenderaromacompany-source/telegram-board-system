# /caspers-telegram-miniapp:sessions

View all active Mini App sessions.

## Instructions

When this command is invoked:

1. Get plugin stats via `plugin.get_stats()`
2. Display the number of active sessions
3. List session details if available

## Example Output

```
👥 Active Sessions: 5

User 12345 - Active since 10:30 AM
User 67890 - Active since 11:15 AM
User 11111 - Active since 12:00 PM
```
